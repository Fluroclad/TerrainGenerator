# Terrain Generator

A terrain generator written in C++ with Vulkan graphics API.

## Requires
* C++17
* Vulkan 1.2

## Dependencies
* Conan (https://conan.io/)

## Compile Steps
1. Create a build directory:

    ```
    $ mkdir build
    $ cd build
    ```

2. Install dependencies:

    ```
    $ conan install ..
    ```
